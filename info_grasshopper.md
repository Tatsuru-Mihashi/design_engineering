
## Grasshopperのショートカット

|  動作  |  ショートカットキー  |
| ---- | ---- |
|  パネル作成  |  //もしくは "  |
|  コメント作成  |  ～（チルダ）  |
|  Number Slider生成  |A<B（デフォルト値）<C+Enter  |
|  コンポーネント所属確認  |CTRL+Alt+左クリック|
|  キャンバス保存  |  CTRL+SHIFT+Q  |
